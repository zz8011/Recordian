# 默认预设：整理口语

请整理以下文本，严格遵循以下规则：

1. **理解文本后去除语气词和重复词**：
   - 删除不必要语气词：嗯、啊、呃、哦、诶、那个、这个、然后、就是、其实、反正等
   - 删除不必要的犹豫和思考时的重复词，如"这个这个"、"就是就是"、"那个那个"等

2. **标点符号使用规则**：
   - 疑问句必须使用问号（？）
   - 非疑问句绝对不能使用问号
   - 根据语气和停顿准确使用逗号、句号、感叹号等

3. **阿拉伯数字判断规则**：
   - 中文数字转换成阿拉伯数字
   - 数学、科学、统计数据使用阿拉伯数字
   - 需要强调或正式场合的数字可使用中文大写
   - 根据上下文语境智能判断

4. **分段和换行**：
   - 长文本根据主题和逻辑关系准确分段
   - 每个独立的观点或话题另起一段
   - 保持段落之间的逻辑连贯性

5. **要点提炼和格式化**（仅针对长文本）：
   - 如果文本较长且缺少结构，提炼关键要点
   - **重要**：保持内容完整，不要过度精炼或删减原意
   - 使用合适的格式（如列表、分段）提升可读性
   - 确保所有重要信息都被保留

6. **输出要求**：
   - 只输出整理后的文本
   - 不要添加任何解释、说明、标记或元信息
